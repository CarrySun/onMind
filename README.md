# onMind
一个vue + vue-router + vuex  + webpack + koa2 + scss + es6 搭建的在线思维导图制作平台

## 功能一览

* 图文编辑（思维导图、文本代码格式化）
* 客户端采用浏览器方式，支持跨平台操作
* 作品的分享、协同编辑等功能

## 开发环境

  win10 

## 技术栈：

vue + vue-router + vuex  + webpack + koa2 + scss + es6

## 总结

### 服务构建

``` bash
# 依赖安装
npm install

# 服务启动
npm run dev

# 打包文件
npm run build

# 文档构建
npm run build --report

# 请求路径
http://127.0.0.1:8000

```
